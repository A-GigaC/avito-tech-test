# Перед запуском
Следует убедиться, что порты 5432 и 8080, на которых будет поднята бд и приложение соответственно, свободны.
# Запуск
Приложение вместе с бд запускаются командой `docker compose up`.
# Дополнительно
В файле `avito-tech-test.bruno.json` лежит bruno-коллекция с примерами запросов на все эндпоинты из спецификации.
IP-адрес, указаный в запросах, следует поменять на тот, который будет актуален для вашей машины (в моем случае это был ip wsl).
